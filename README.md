# weather_weasel_prototype

A prototype of a Ruby wrapper for the Weather Underground API

## Installation

Add this line to your application's Gemfile:

    gem 'weather_weasel'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install weather_weasel
